# Task#06 Static Site

This project is a static website and is ready to deploy.

## What is included
- Main page: index.html
- Local assets: the bundled CSS/JS/image files in the Colorlib assets folder
- GitHub Pages deployment workflow: .github/workflows/deploy.yml

## Deploy options
### GitHub Pages
1. Push this folder to a GitHub repository.
2. Open the repository Settings > Pages.
3. Choose GitHub Actions as the source.
4. The workflow will publish the site automatically.

### Netlify or Vercel
Upload the project folder as a static site. No build step is required.
